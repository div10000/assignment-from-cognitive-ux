Design & Implementation Notes

  • Background Styles & Inheritance: On the live webpage, the textured peach background is inherited from a parent wrapper (e.g., the global <body> or a parent container) or styled via pseudo-elements on the parent, as the section.exclusive element itself is transparent. For this standalone replica, the gradient has been styled directly on the section container for self-containment.
  • Swiper.js Stacking & Defaults: The original layout relies on the external Swiper.js library's default styles (such as its default blue color #007aff for navigation elements and specific positioning of bullet points). Since the task requirements forbid external libraries and frameworks, these transitions, 3D rotations, and pagination states are simulated using native CSS transforms and vanilla JS.
  • Font Rendering: Differences in rendering may exist depending on local system fonts and font-smoothing, but we have aligned the typeface to the custom Mulish font family loaded from Google Fonts as observed on the original webpage.
